# shadcnui/.README.md

- Default directory for shadcn/ui components, new shadcn component installs will be automatically added to this directory.
- Theme can be found in `src/css/global.css`
- Configuration can be found in `components.json`
- Our shadcn/ui design system is based on [Base UI](https://base-ui.com/) primitives rather than Radix.
- We also use the [ReUI](https://reui.io/) registry as an extension to shadcn/ui

---

It is highly discouraged to modify the core shadcn/ui components, as versioning updates may override changes _(usually found at the bottom of each component's shadcn/ui documentation page)_.

Instead, opt to import these components within your own, and apply overrides using available props.

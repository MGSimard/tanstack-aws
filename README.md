## Tech Stack

- AWS stuff

- [React](https://react.dev/) - Frontend Library
- [TypeScript](https://www.typescriptlang.org/) - Fullstack Language

- [TanStack Start](https://tanstack.com/start/latest) - Fullstack Framework
- [TanStack Router](https://tanstack.com/router/latest) - Type-Safe routing & app-wide route context & state management
- [TanStack Query](https://tanstack.com/query/latest) - Data fetching & Async State Management
- [TanStack DB](https://tanstack.com/db/latest) - Wrapper for reactive live queries & optimistic mutations
- [TanStack Store](https://tanstack.com/store/latest) - State management automatically leveraged by TanStack libraries
- [TanStack AI](https://tanstack.com/ai/latest) - AI SDK
- [TanStack Form](https://tanstack.com/form/latest) - TypeSafe form state management
- [TanStack Table](https://tanstack.com/table/latest) - Data tables/grids
- [TanStack Devtools](https://tanstack.com/devtools/latest) - Developer tools for TanStack Query and TanStack DB

- [Drizzle](https://orm.drizzle.team/) - Database ORM (Maybe)

- [Tailwind CSS](https://tailwindcss.com/) - CSS Framework
- [Base UI](https://base-ui.com/) - Primitive UI components
- [shadcn/ui](https://ui.shadcn.com/) - Component library & design system
- [ReUI](https://reui.io/) - shadcn/ui extension

- [Zod](https://zod.dev/) - Schema validation
- [Oxlint](https://oxc.rs/docs/guide/usage/linter.html) - Rust-based Linter
- [Oxfmt](https://oxc.rs/docs/guide/usage/formatter.html) - Rust-based Formatter
